# PROJECT CONTEXT

## Proje Tanımı

Bu proje bir makine ekipman kiralama ve takip sistemidir.

## Teknolojiler

* Backend: Flask (Python)
* Frontend: Flutter
* Database: PostgreSQL
* API: RESTful

## Kullanıcı Rolleri

* Admin
* Operasyon
* Bakım

## Temel Modüller

* Makine yönetimi
* Kiralama sözleşmeleri
* Bakım kayıtları
* Yedek parça yönetimi

## Kodlama Kuralları

* Modüler yapı zorunlu
* Fonksiyonlar küçük ve okunabilir olmalı
* Gereksiz abstraction yok
* Açık ve anlaşılır isimlendirme

## API Standartları

* JSON response
* HTTP status code doğru kullanılmalı
* Endpoint isimleri tutarlı olmalı

## Yasaklar

* Spaghetti code yok
* Tek dosyada aşırı logic yok
* Hardcoded değerlerden kaçınılacak

## Notlar

* Sistem çok kullanıcılı çalışacak
* Gerçek zamanlı veri tutarlılığı önemli
